Html
HTML stands for HyperText Markup Language, and it is the standard markup language used to create and structure the skeletal framework of web pages. Rather than a traditional programming language that executes computational logic, HTML serves as a structural formatting system.
Css
CSS stands for Cascading Style Sheets. It is a rule-based stylesheet language used to control the visual presentation, formatting, and layout of a document written in HTML or XML. Along with HTML and JavaScript, CSS is a core technology of the World Wide Web.
Javascript
JavaScript (JS) is a lightweight, interpreted programming language used primarily to create dynamic and interactive content on websites. Alongside HTML (which structures web content) and CSS (which styles it), JavaScript serves as the core layer of standard web technologies.
React js
React (also known as React.js or ReactJS) is a free, open-source front-end JavaScript library used specifically for building dynamic and interactive user interfaces (UIs) for web applications.
